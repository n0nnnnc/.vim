## Requirements:
* vim
* vim-gtk3 (gvim)

## Installation:
`git clone git@github.com:n0nnnnc/.vim.git ~/.vim`

## Create symlinks:
```
ln -s ~/.vim/vimrc ~/.vimrc
ln -s ~/.vim/gvimrc ~/.gvimrc
```

## Switch to the ~/.vim directory, and fetch submodules:
```
cd ~/.vim
git submodule init
git submodule update
```
